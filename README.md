# Rule Radar

**Jurisdiction-based legal intelligence platform**

Search, understand, compare, and track laws across city, county, state, and federal levels through searchable records, citations, plain-language summaries, version history, and alerts.

## Stack
- **Framework:** Next.js 14 (App Router)
- **Database:** Supabase (PostgreSQL)
- **Deployment:** Vercel
- **Styling:** Tailwind CSS + CSS Custom Properties

## Architecture
- 34 database tables across 11 domains
- Full-text search on law records
- Row-level security policies
- 37 planned screens across 4 surfaces (public, pro, editorial, admin)
